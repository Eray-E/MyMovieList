# MyMovieList
Teilnehmer: Eray Elman , Matrikelnummer: s0598269

Eine Movie Liste in der man Filme suchen,hinzufügen,favorisieren,löschen,sortieren, bewerten, den Status anpassen kann und seine Statistiken nachvollziehen kann.

Durch die Eingabe in der Suchleiste werden Titel von Filmen vorgeschlagen die man auswählen kann. Das Jahr und Genre wird automatisch angepasst.
Diese stammen aus einer externen Movie API von https://www.omdbapi.com/
Mit Add Movie wird der Film in die Liste hinzugefügt dort kann man links auf den linken Stern klicken um es zu favorisieren.
Rechts neben Genre des Filmes wird ein Status Dropdown erstellt in der man auswählen kann ob man den Film Completed (fertig geguckt hat), Dropped(abgebrochen hat) oder Plan to watch (es noch vorzuhaben).
Neben den Status Drop Down gibt es ein Rating Drop Down dort kann man seinen Film von 1-10 bewerten.
Falls man einen Film von seiner Liste löschen will, klickt man auf den Delete Icon neben den Ratin Drop Down.
Unter der Suchleiste befindet sich ein Sort By Drop Down mit der man die Filme nach Title (alphabetisch), nach Rating (aufsteigend) , nach Jahr (aufsteigend) und nach favoriten
Gegenüber befindet sich ein Stats Button die eine neue Seite öffnet wo alle Statistiken aufgezeigt werden z.B. Wie viele Filme man Insgesammt auf der Liste hat, wie viele man completed hat , die duchschnittsbewertung etc..
Klickt man oben rechts auf Back to Watchlist kommt man wieder zurück zu seiner Movie Liste.
